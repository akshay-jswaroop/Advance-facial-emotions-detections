# 😃 Real-Time Facial Emotion Detection

## 📌 Overview
This project uses Deep Learning and Computer Vision to detect human emotions from facial expressions in real-time.

## 🚀 Features
- Detect emotions like Happy, Sad, Angry, Surprise
- Real-time webcam detection
- Image-based emotion prediction
- Deep Learning model integration

## 🛠️ Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib
